# Options Premium


# 期权溢价
The price paid to acquire the option. Also known simply as option price. Not to be confused with the         strike price. Market price, volatility and time remaining are the primary forces determining the premium.\xc2\xa0 There are two components to the options premium and they are intrinsic value and time value.

b'\xe8\x8e\xb7\xe5\xbe\x97\xe6\x9c\x9f\xe6\x9d\x83\xe6\x89\x80\xe6\x94\xaf\xe4\xbb\x98\xe7\x9a\x84\xe4\xbb\xb7\xe6\xa0\xbc\xe3\x80\x82'
## Intrinsic Value


## 内在价值
The intrinsic value is determined by the difference between the current trading price and the strike price. Only in-the-money options    have intrinsic value. Intrinsic value can be computed for in-the-money options by taking the difference between the strike price and the current trading price. Out-of-the-money options have no intrinsic value.

b'\xe5\x86\x85\xe5\x9c\xa8\xe4\xbb\xb7\xe5\x80\xbc\xe7\x94\xb1\xe5\xbd\x93\xe5\x89\x8d\xe4\xba\xa4\xe6\x98\x93\xe4\xbb\xb7\xe6\xa0\xbc\xe4\xb8\x8e\xe6\x89\xa7\xe8\xa1\x8c\xe4\xbb\xb7\xe6\xa0\xbc\xe4\xb9\x8b\xe9\x97\xb4\xe7\x9a\x84\xe5\xb7\xae\xe5\xbc\x82\xe5\x86\xb3\xe5\xae\x9a\xe3\x80\x82'
## Time Value


## 时间价值
An option's time value is dependent upon the length of time remaining to exercise the option, the moneyness of the option, as well as the volatility of the underlying security's market price. 

b'\xe6\x9c\x9f\xe6\x9d\x83\xe7\x9a\x84\xe6\x97\xb6\xe9\x97\xb4\xe4\xbb\xb7\xe5\x80\xbc\xe5\x8f\x96\xe5\x86\xb3\xe4\xba\x8e\xe8\xa1\x8c\xe4\xbd\xbf\xe6\x9c\x9f\xe6\x9d\x83\xe7\x9a\x84\xe5\x89\xa9\xe4\xbd\x99\xe6\x97\xb6\xe9\x97\xb4\xe9\x95\xbf\xe5\xba\xa6\xef\xbc\x8c\xe6\x9c\x9f\xe6\x9d\x83\xe7\x9a\x84\xe8\xb4\xa7\xe5\xb8\x81\xe4\xbb\xa5\xe5\x8f\x8a\xe6\xa0\x87\xe7\x9a\x84\xe8\xaf\x81\xe5\x88\xb8\xe5\xb8\x82\xe5\x9c\xba\xe4\xbb\xb7\xe6\xa0\xbc\xe7\x9a\x84\xe6\xb3\xa2\xe5\x8a\xa8\xe6\x80\xa7\xe3\x80\x82'
The time value of an option decreases as its         expiration date approaches and becomes worthless after that date. This phenomenon    is known as time decay. As such,    options are also wasting assets.

b'\xe6\x9c\x9f\xe6\x9d\x83\xe7\x9a\x84\xe6\x97\xb6\xe9\x97\xb4\xe4\xbb\xb7\xe5\x80\xbc\xe9\x9a\x8f\xe7\x9d\x80\xe5\x88\xb0\xe6\x9c\x9f\xe6\x97\xa5\xe7\x9a\x84\xe4\xb8\xb4\xe8\xbf\x91\xe8\x80\x8c\xe5\x87\x8f\xe5\xb0\x91\xef\xbc\x8c\xe5\xb9\xb6\xe5\x9c\xa8\xe8\xaf\xa5\xe6\x97\xa5\xe6\x9c\x9f\xe4\xb9\x8b\xe5\x90\x8e\xe5\x8f\x98\xe5\xbe\x97\xe6\xaf\xab\xe6\x97\xa0\xe4\xbb\xb7\xe5\x80\xbc\xe3\x80\x82'
For in-the-money options, time value can be calculated by subtracting the intrinsic value from the option price. Time value decreases as the option goes deeper into the money. For out-of-the-money options, since there is zero intrinsic value, time value = option price.    

b'\xe5\xaf\xb9\xe4\xba\x8e\xe4\xbb\xb7\xe5\x86\x85\xe6\x9c\x9f\xe6\x9d\x83\xef\xbc\x8c\xe5\x8f\xaf\xe4\xbb\xa5\xe9\x80\x9a\xe8\xbf\x87\xe4\xbb\x8e\xe6\x9c\x9f\xe6\x9d\x83\xe4\xbb\xb7\xe6\xa0\xbc\xe4\xb8\xad\xe5\x87\x8f\xe5\x8e\xbb\xe5\x86\x85\xe5\x9c\xa8\xe4\xbb\xb7\xe5\x80\xbc\xe6\x9d\xa5\xe8\xae\xa1\xe7\xae\x97\xe6\x97\xb6\xe9\x97\xb4\xe4\xbb\xb7\xe5\x80\xbc\xe3\x80\x82'
        Typically, higher volatility give rise to higher time value. In general, time value        increases as the uncertainty of the option's value at expiry increases.

b'\xe9\x80\x9a\xe5\xb8\xb8\xef\xbc\x8c\xe8\xbe\x83\xe9\xab\x98\xe7\x9a\x84\xe6\xb3\xa2\xe5\x8a\xa8\xe6\x80\xa7\xe4\xbc\x9a\xe4\xba\xa7\xe7\x94\x9f\xe8\xbe\x83\xe9\xab\x98\xe7\x9a\x84\xe6\x97\xb6\xe9\x97\xb4\xe4\xbb\xb7\xe5\x80\xbc\xe3\x80\x82'
Time value of call options on high cash dividend stocks can get discounted while        similarly, time value of put options can get inflated. For more details on the effect        of dividends on option pricing,             read this article.
