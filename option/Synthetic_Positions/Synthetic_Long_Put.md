# Synthetic Long Put


# 合成长放
A synthetic long put is created when short stock position is combined with a long call of the same series.

b'\xe5\xbd\x93\xe7\xa9\xba\xe5\xa4\xb4\xe8\x82\xa1\xe7\xa5\xa8\xe5\xa4\xb4\xe5\xaf\xb8\xe4\xb8\x8e\xe5\x90\x8c\xe4\xb8\x80\xe7\xb3\xbb\xe5\x88\x97\xe7\x9a\x84\xe9\x95\xbf\xe6\x9c\x9f\xe7\x9c\x8b\xe6\xb6\xa8\xe6\x9c\x9f\xe6\x9d\x83\xe7\x9b\xb8\xe7\xbb\x93\xe5\x90\x88\xe6\x97\xb6\xef\xbc\x8c\xe4\xbc\x9a\xe5\x88\x9b\xe5\xbb\xba\xe4\xb8\x80\xe4\xb8\xaa\xe5\x90\x88\xe6\x88\x90\xe5\xa4\x9a\xe5\xa4\xb4\xe7\x9c\x8b\xe8\xb7\x8c\xe6\x9c\x9f\xe6\x9d\x83\xe3\x80\x82'
The synthetic long put is so named because the established position has the same profit potential as long put.
